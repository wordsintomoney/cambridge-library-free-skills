# The Cambridge Library Free Skills

A curated collection of free, ready-to-use AI agent skills designed to help you get real work done with AI.

These are practical, reusable systems built for creators, small business owners, and anyone who wants to move faster, think clearer, and produce better results using AI.

## What You’ll Find Here

This repository includes a growing set of agent skills you can use immediately:

- Writing and content creation
- Marketing and messaging
- Product and offer development
- Idea generation and validation
- Workflow and productivity support

Each skill is designed to be:

- Clear
- Practical
- Easy to use
- Easy to adapt to your own needs

---

## How to Use These Skills

Each skill lives inside its own folder and typically includes:

- A system prompt or core instruction file  
- Usage guidance  
- Example inputs and outputs (when applicable)

To use a skill:

1. Open the skill folder
2. Copy the core instructions or prompt
3. Paste it into your AI tool of choice (ChatGPT, Claude, etc.)
4. Follow the usage notes provided

That’s it.
