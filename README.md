---

## How to Use These Skills

Each skill lives inside its own folder and typically includes:

- A system prompt or core instruction file  
- Usage guidance  
- Example inputs and outputs (when applicable)

To use a skill:

1. Open the skill folder
2. Copy the core instructions or prompt
3. Paste it into your AI tool of choice (ChatGPT, Claude, etc.)
4. Follow the usage notes provided

That’s it.
